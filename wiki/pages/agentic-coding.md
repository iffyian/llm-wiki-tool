# Agentic Coding

## Summary

Agentic coding은 LLM을 단순 코드 생성 도구가 아니라 분석, 계획, 구현, 검토를 수행하는 시스템 구성요소로 활용하는 방식입니다.

## Key Points

- 코딩은 SDLC 전체 중 일부 활동입니다.
- 빠른 프로토타이핑만으로 좋은 시스템이 완성되지는 않습니다.
- 목표, 요구사항, 경계, 제약조건을 함께 분석해야 합니다.
- Agentic coding은 user request를 작은 task로 분해하고 검증 가능한 workflow로 처리합니다.
