# Harness, MCP, Automation

## Summary

Harness engineering, MCP, hook은 LLM agent가 반복 가능한 방식으로 안전하게 작업하도록 돕는 실행 환경과 연결 구조입니다.

## Key Points

- Harness engineering은 도구, 데이터, 검증 절차, 안전장치를 구조적으로 설계합니다.
- Skill은 agent가 반복해서 사용할 수 있는 작업 절차입니다.
- MCP는 agent와 외부 시스템을 표준화된 방식으로 연결합니다.
- Hook은 agent 생애주기의 특정 시점에 자동으로 실행되는 script입니다.
- Hook은 log 기록, knowledge base 확인, 종료 전 검증 등에 사용할 수 있습니다.
